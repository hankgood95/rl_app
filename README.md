# PTE RL 템플릿 퀴즈

PTE Academic **Retell Lecture(RL)** 템플릿 암기를 위한 퀴즈 웹앱입니다.

🔗 **배포 URL**: https://hankgood95.github.io/rl_app/

---

## 주요 기능

| 화면 | 기능 |
|------|------|
| 🏠 홈 | 버전 / 모드 / 난이도 설정 |
| 🎯 퀴즈 | 타이핑 또는 드롭다운 4지선다 빈칸 채우기 |
| 📖 공부하기 | 플래시카드 앞/뒤 뒤집기, 랜덤 꿀팁 |
| 📊 통계 | 버전별 누적 정답률 바 차트 |
| 📝 오답노트 | 틀린 표현 목록 (많이 틀린 순 정렬) |
| 💡 꿀팁 전체보기 | PTE RL 학습 꿀팁 8개 |

## 템플릿 버전

- **버전1 · 표준형** — 기본 구조의 정형화된 템플릿
- **버전2 · 흥미강조형** — interesting / notable 등 흥미 표현 강조
- **버전3 · 통찰강조형** — insightful / addressed / emphasized 등 통찰 표현 강조

## 난이도

| 난이도 | 빈칸 수 |
|--------|---------|
| 입문 | 3개 |
| 중급 | 5개 |
| 고급 | 8개 |
| 전체 | 전부 |

## 기술 스택

- 순수 HTML / CSS / JavaScript (외부 라이브러리 없음)
- 단일 파일(`index.html`)
- 다크 테마, 모바일 반응형
- `localStorage` 기반 통계 및 오답노트 영구 저장
- GitHub Pages 배포
